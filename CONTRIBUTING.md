# 🎮 Contribuindo para Sky Bird

Obrigado pelo interesse em contribuir! Este é um projecto simples e qualquer ajuda é bem-vinda.

## 🐛 Reportar Bugs

Encontrou um bug? Abra uma [issue](https://github.com/seu-usuario/jogo-fly-bird/issues) com:

- **Título descritivo** do problema
- **Passos para reproduzir** o comportamento
- **Comportamento observado** vs **comportamento esperado**
- **Capturas de tela** se aplicável
- **Ambiente:** navegador, OS, versão

## 🔧 Como Contribute Código

### Setup Inicial

```bash
git clone https://github.com/seu-usuario/jogo-fly-bird.git
cd jogo-fly-bird
```

Não há dependências! Basta abrir `index.html` no navegador.

### Fluxo de Desenvolvimento

1. **Fork** o repositório
2. **Crie uma branch** para sua feature (`git checkout -b feature/amazing-feature`)
3. **Commit** as mudanças (`git commit -m 'Add some amazing feature'`)
4. **Push** para a branch (`git push origin feature/amazing-feature`)
5. **Abra um Pull Request**

### Padrões de Código

- **JavaScript:** Use `const` por padrão, `let` se necessário reatribuir
- **Comentários:** Use JSDoc para funções públicas
- **Nomes:** CamelCase para variáveis, snake_case para IDs HTML
- **Espaçamento:** 2 espaços, sem tabs
- **Sem dependências externas:** Mantenha vanilla JS puro

### Testes

Teste no:
- Desktop (Chrome, Firefox, Safari, Edge)
- Mobile (iPhone, Android)
- Modo responsivo de ferramentas do desenvolvedor

### Mensagens de Commit

Use imperative mood ("Add feature" not "Added feature")

```
Capitanear o título com um verbo de ação
- Descrever mudanças de forma concisa
- Referenciar issues: Closes #123
```
